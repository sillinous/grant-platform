# UNLESS — Grant Lifecycle Platform v4.0

> 31 modules · 23+ live APIs · AI-powered intelligence · $0/month

A comprehensive grant management platform built for entrepreneurs, nonprofits, and small organizations. Discover federal grants, write proposals with AI, track compliance, manage awards, and optimize your portfolio — all in one place.

## Modules

| Group | Modules |
|---|---|
| **Core** | Dashboard, Discovery, Pipeline, Calendar, Deadline Watchdog |
| **Analysis** | RFP Parser, Match Scorer, Match Alerts |
| **Writing** | AI Drafter, Narrative Scorer, Section Library, Letter Generator, Census Narratives |
| **Documents** | Budget Builder, Document Vault |
| **Management** | Compliance Tracker, Action Plan, Award Management, Collaboration Hub |
| **Intelligence** | Financial Projector, Funding Forecast, Relationship Map, Peer Prospecting, Funder Research, Portfolio Optimizer, Win/Loss Analysis, Impact Portfolio |
| **Output** | Report Generator, Activity Log |

## Live API Integrations

- **Grants.gov** — Federal grant search & opportunity details
- **USASpending.gov** — Award data, recipient search, spending trends
- **Census Bureau ACS** — Demographic data for Statement of Need narratives
- **Regulations.gov** — Regulatory intelligence & comment tracking
- **SAM.gov** — Entity verification & registration status
- **Anthropic Claude** — AI drafting, scoring, analysis, and chat (user's own key)
