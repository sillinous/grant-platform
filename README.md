# UNLESS — Grant Lifecycle Platform v5.0

> 39 modules · 23+ live APIs · AI-powered intelligence · $0/month

A comprehensive grant management platform built for entrepreneurs, nonprofits, and small organizations. Discover federal grants, write proposals with AI, track compliance, manage awards, and optimize your portfolio — all in one place.

## 39 Modules Across 8 Groups

| Group | Modules |
|---|---|
| **Core (6)** | Dashboard, Discovery, Pipeline, Calendar, Deadline Watchdog, Intelligence Feed |
| **Analysis (4)** | RFP Parser, Match Scorer, Match Alerts, Readiness Assessment |
| **Writing (6)** | AI Drafter, Narrative Scorer, Section Library, Letter Generator, Census Narratives, Document Assembler |
| **Documents (3)** | Budget Builder, Document Vault, Grant Templates |
| **Management (6)** | Compliance Tracker, Action Plan, Award Mgmt, Outcome Tracker, Collaboration Hub, SAM Registration Wizard |
| **Intelligence (9)** | Financial Projector, Funding Forecast, AI Strategic Advisor, Relationship Map, Peer Prospecting, Funder Research, Portfolio Optimizer, Win/Loss Analysis, Impact Portfolio |
| **Output (3)** | Report Generator, Export Center, Activity Log |
| **System (1)** | Settings |

## Live API Integrations

- **Grants.gov** — Federal grant search & opportunity details
- **USASpending.gov** — Award data, recipient search, spending trends
- **Census Bureau ACS** — Demographic data for Statement of Need narratives
- **Regulations.gov** — Regulatory intelligence & comment tracking
- **SAM.gov** — Entity verification & registration status
- **Anthropic Claude** — AI drafting, scoring, analysis, and chat
